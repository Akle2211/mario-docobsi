# mario-docobsi
Public repository of documentation regarding a Mario-like game development in Godot.

# Installation Obsidian
- installer [Obsidian](https://obsidian.md)
-  dans une console ouverte dans un dossier, taper ```git clone https://github.com/Akle2211/mario-docobsi.git``` 
- ouvrir ce dossier dans Obsidian (donc faire open vault)
- installer le plugin [Git](obsidian://show-plugin?id=obsidian-git) dans Obsidian pour ce dossier mario-docobsi
- lors de votre premier git push, il faudras un token comme password. Demander le à Luca, ou trouvez le sur le discord, canal général :)
